# スマホPWA版

iPhone / Androidのブラウザで動くLINEスタンプ画像整形ツールです。画像は外部へ送信せず、端末内で処理します。

## 起動

PWA機能はHTTP/HTTPS配信が必要です。開発用にはこのフォルダで次を実行します。

```bash
python -m http.server 8080
```

同じPCでは `http://localhost:8080` を開きます。スマホから試す場合は、PCとスマホを同じWi-Fiに接続し、PCのローカルIPを使って `http://PCのIP:8080` を開きます。

本番利用では、このフォルダをGitHub Pages、Cloudflare Pages、NetlifyなどのHTTPS対応静的ホスティングへ配置してください。

## ホーム画面への追加

- iPhone: Safariの共有ボタンから「ホーム画面に追加」
- Android: Chromeのメニューから「アプリをインストール」または「ホーム画面に追加」

## 対応機能

- 1枚のグリッド画像、または複数画像から入力
- 8 / 16 / 24 / 32 / 40個
- 白背景除去、余白トリミング、padding
- 370x320以内の透過PNG
- main.png 240x240、tab.png 96x74
- ZIPとreport.txtを端末内で生成
- 初回表示後のオフライン起動
- 起動時のバージョン確認と自動更新
- 単体画像をスタンプ、メイン、トークルームタブ画像へリサイズしてPNG保存
- 8コマ分割後の各スタンプに同じ文字を合成

## 注意

- iPhoneではZIPは通常「ファイル」アプリのダウンロードへ保存されます。
- 大きな画像を40枚処理すると、古い端末では時間やメモリを多く使います。
- 白背景除去は色しきい値方式のため、白い絵柄も透明になる場合があります。
- ストア配布版は、このPWAをCapacitorでラップする形で追加できます。
